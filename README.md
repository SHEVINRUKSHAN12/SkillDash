# SkillDash – Service Marketplace Web App

SkillDash is a full-stack web application designed to connect customers with skilled service providers. It streamlines the process of job posting, service offering, communication, and feedback in a user-friendly and efficient platform.

---

## 🚀 Features

### 🧩 Sprint 1 – Backend Development
- Admin System Authentication, dashboard control, user and provider management
- Customer & Provider Accounts Registration, login, profile management, credential uploads
- Job Management Create, edit, delete, and search job listings
- Service Management Service creation, editing, visibility toggling
- Chat System Real-time chat using Socket.IOFirebase, with notification & persistence
- Feedback & Ratings 5-star rating system with review moderation
- Analytics Dashboard Backend setup for tracking usage & performance

### 🎨 Sprint 2 – Front-End Integration
- Responsive UI For customers, providers, and admin dashboards
- Interactive Chat Interface
- Real-Time Job Matching Matches jobs with providers based on category, location, and availability
- Ratings & Reviews Display
- Notification Preferences & Settings

---

## 🔧 Technologies Used
- Frontend React.js, Tailwind CSS
- Backend Node.js, Express.js
- Database MongoDB
- Real-time Socket.IO  Firebase
- Authentication JWT
- DevOps Azure DevOps (Sprint Planning & Task Tracking)
